🚢 Batalla Naval
📖 Descripción

Este proyecto consiste en una implementación básica del clásico juego Batalla Naval desarrollada en lenguaje C para ejecutarse por consola.

El objetivo principal fue recrear las reglas fundamentales del juego, permitiendo al usuario interactuar con un tablero,
realizar disparos y detectar impactos o fallos hasta hundir todos los barcos enemigos.

✨ Funcionalidades
Representación del tablero por consola.
Ubicación de barcos en posiciones válidas.
Sistema de disparos mediante coordenadas.
Detección de impactos y disparos fallidos.
Actualización del estado del tablero después de cada turno.
Verificación de condiciones de victoria.
🛠️ Tecnologías utilizadas
Lenguaje C
Compilador GCC / Code::Blocks
Estructuras de datos y lógica de programación
🎯 Objetivos del proyecto

Durante el desarrollo de este proyecto se aplicaron conceptos de:

Programación estructurada.
Uso de matrices bidimensionales.
Manejo de funciones.
Validación de entradas del usuario.
Implementación de lógica de juego.
